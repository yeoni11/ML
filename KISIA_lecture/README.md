# 퍼셉트론 실습 (101_perceptron)
KISIA - AI 보안 개론 수업에서 퍼셉트론의 기본 개념을 실습한 노트북입니다.


## 다룬 내용
> (1) 퍼셉트론 기본 구조 (가중치, 바이어스)  
> (2) AND, OR, NAND 게이트 단층 퍼셉트론 구현  
> (3) 단층 퍼셉트론의 XOR 한계 (선형 분리 불가)  

> (4) 다층 퍼셉트론으로 XOR 구현1 (OR + NAND 조합)  
> (5) 다층 퍼셉트론으로 XOR 구현2 (NAND 4개)  

## 실행 방법
> (1) Google Colab에서 `101_perceptron.ipynb` 열기  
> (2) 셀 순서대로 실행
